4.2 Root Cause
After investigation, the reason for the mismatch is SEBI's mutual fund recategorisation circular from 2018, which required all AMCs to rename and restructure their schemes. As a result:
— SBI Bluechip Fund was renamed to SBI Large Cap Fund
— ICICI Prudential Bluechip Fund was renamed to ICICI Prudential Large Cap Fund
— Axis Bluechip Fund was renamed to Axis Large Cap Fund
— Kotak Bluechip Fund was renamed to Kotak Large Cap Fund
— AMFI scheme codes were also reassigned in several cases during this process
The mfapi.in API always reflects the current AMFI registry. The codes in the documentation were based on an older version of the registry and are now outdated.


4.3 How It Was Fixed
The mfapi.in search endpoint was used to look up schemes by name and find correct current codes. Each code was then verified by directly calling the API and checking the scheme name in the response. The corrected codes used are:

8. Key Observations
— AMFI scheme codes should never be hardcoded — they must be verified at runtime against the live API
— SEBI's 2018 recategorisation caused widespread renaming of mutual fund schemes across all AMCs
— The /mf/{code}/latest endpoint is much faster than /mf/{code} for bulk code verification
— All 40 AMFI codes in the provided fund_master.csv matched nav_history.csv with zero discrepancies
— The mfapi.in API is reliable and returns well-structured JSON with both metadata and full NAV history
